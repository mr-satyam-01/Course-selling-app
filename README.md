# Course-selling-app
Course Selling App Backend

A Node.js backend API for a course-selling platform where users can sign up, log in, and purchase courses. Admins can create and manage courses. This project demonstrates authentication, authorization, database design, and REST API development using modern backend tools.

🚀 Tech Stack
	•	Node.js
	•	Express.js
	•	MongoDB
	•	Mongoose
	•	JWT Authentication
	•	Bcrypt (Password Hashing)
	•	Zod (Input Validation)


Features
User
	•	User signup
	•	User login with JWT authentication
	•	View purchased courses
Admin
	•	Admin signup and login
	•	Create new courses
	•	Delete courses
	•	Manage course content
Security
	•	Password hashing with bcrypt
	•	Authentication using JWT tokens
	•	Request validation using Zod
